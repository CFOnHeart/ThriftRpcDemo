# ThriftRpcDemo
a hello world demo: Thrift to Java in maven project
### enviroment
thrift: 0.9.3

maven: 3.5.0

jdk: 1.8

Intellij

### steps
```bash
# bash
git clone ${this respository}
cd ThriftRpcDemo
cd HelloService
mvn clean install  # package jar at local
```

intellij open project HelloServer

excute Server.main

excute Client.main
